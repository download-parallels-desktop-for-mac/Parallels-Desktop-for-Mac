# 🧠 What is Parallels Desktop?

Parallels Desktop is virtualization software that creates a **virtual machine (VM)** on macOS. A virtual machine acts like a separate computer running inside your Mac.

Instead of restarting your Mac to switch operating systems, Parallels allows you to run:

- Windows
- Linux
- Ubuntu
- Fedora
- Kali Linux
- Older macOS versions

directly alongside macOS.

This means you can use macOS and Windows applications at the same time on a single screen.
